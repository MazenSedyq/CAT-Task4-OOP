# CAT-Task4-OOP
a console-based Superhero Battle Arena 

## How to play (controls)
- On startup choose:
  - `[1] NEW GAME` — create fighters and start battle
  - `[2] HOW TO PLAY` — view instructions
- During a round each fighter chooses:
  - `[1] ATTACK` — attempt to deal damage (may miss based on Accuracy)
  - `[2] HEAL` — recover HP (based on Recover stat)
- Game ends when one hero's Health reaches 0.
